# Life Tracker

Capture and visualize the data that matters in your life. This Obsidian plugin adds custom Base View types that transform your tracked data into beautiful visualizations.

## Key Features

- **12 Visualization Types**: Heatmap, Line, Bar, Area, Pie, Doughnut, Radar, Polar Area, Scatter, Bubble, Tag Cloud, Timeline
- **Overlay Charts**: Combine multiple properties on a single chart to visualize correlations
- **Property Capture**: Quick dialog to fill properties with carousel navigation and batch processing
- **Grid View**: Card-based editing interface for managing properties across multiple notes
- **Customization**: Per-view settings, time frames, color schemes, scales, and reference lines
- **Interactive Elements**: Click chart elements to open source notes, hover for detailed tooltips

## Quick Start

1. Install the plugin from Obsidian Community Plugins
2. Create a Base in your vault
3. Open the Base and add property columns you want to track
4. Click the view selector and choose **Life Tracker**
5. Click on unconfigured property cards to choose a visualization type

## Views

### Life Tracker View

A grid-based dashboard showing visualizations for each property in your Base. Configure time granularity, date ranges, and individual visualization settings.

### Life Tracking Grid

A card-based editing interface for quickly filling in property values across multiple notes. Ideal for batch data entry.
